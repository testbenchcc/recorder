## Recorder interface

### Overview

#### Hardware

This is a RP Zero 2 W paired with a KEYESTUDIO ReSpeaker 2-Mic Pi HAT V1.0. They are mounted on a Geekworm X306 V1.5 UPS Expansion Board.

#### Purpose

Serve as a control and status interface for the KEYESTUDIO ReSpeaker 2-Mic Pi HAT V1.0.

### Pages

#### Home

Initial interface; Control and status are the purpose for this page. The user shoulde be able to imidiatly control the device and know what is happening with it. This includes how much space is left for recording (in time unit).

#### Recording

Management of existing recordings, allow removal, renaming, and listening.

#### Style and formating

Bootstrap 5.0
